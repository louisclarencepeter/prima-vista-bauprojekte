# Contributing to Prima Vista Bauprojekte

Thank you for your interest in contributing! We welcome contributions from the community.

## Reporting Bugs

If you find a bug, please open an issue with a clear description, steps to reproduce it, and the expected versus actual behavior.

## Suggesting Features

Feature requests are welcome. Please open an issue describing what you'd like and why it would be useful.

## Submitting Pull Requests

Fork the repository, create a feature branch, commit your changes with clear messages, push to your fork, then open a pull request against the `main` branch.

## Code Style

Follow the existing code conventions. Use TypeScript where applicable. Run `npm run lint` before submitting.

## Development Setup

Run `npm install` then `npm run dev` to start the local development server.

Open a discussion or contact the maintainer via GitHub at @louisclarencepeter.
